Bestanden maken

Op school maken we gebruik van het gratis website Replit . Je moet enkel inloggen met je school account en dan kan je aan de slag! Dit is een tekst editor die je kunt gebruiken om websites te typen. In deze opdracht ga je een opzet maken om jouw eertse pagina te maken.

Opdracht

  1. Maak links boven een nieuwe Repl aan.
  2. Kies voor de template HTML, CSS, JS en geef het een goede naam
  3. Duw op creÃ«er
  4. Je hebt nu drie bestanden links, een tekst editor in het midden en de output rechts
    1. De naam van het bestand staat voor de '.' en het type bestand erachter (let hier altijd goed op)
    2. De eerste pagina noemt altijd index.html, daarna moet je zelf logische namen verzinnen voor de andere pagina's. De regels leer je later in naamgeving.
    3. Een HTML bestand dient voor de tekst, een CSS bestand dient voor de stijl. JS bestanden gaan we niet gebruiken maar dienen voor interactie (je kan deze dus verwijderen)
