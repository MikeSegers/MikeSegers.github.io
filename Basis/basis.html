Bestanden maken

Op school maken we gebruik van het gratis programma Notepad++ voor Windows. Ook staat tegenwoordig Visual Studio Code op de computers, en anders kun je deze downloaden. Dit is een tekst editor die je kunt gebruiken om websites te typen. In deze opdracht ga je een opzet maken om jouw bestanden op te slaan.

Opdracht

  1. Open de documenten map en maak daar een nieuwe map aan. Noem hem bijvoorbeeld website_klas_4.
  2. Klik op de windows toets en type Notepad++ en open het programma.
  3. Klik op ctrl en n om een nieuw bestandje te krijgen.
  4. Klik op ctrl, alt en s om het bestand op te slaan.
    1. Zoek naar de map die je net hebt gemaakt voor jouw website.
    2. De eerste pagina kan het beste index.html worden genoemd, daarna moet je zelf logische namen verzinnen. De regels leer je later in naamgeving.
    3. De bestandstype moet een hypertext markup language file zijn, anders snapt de computer niet dat het een website is.
