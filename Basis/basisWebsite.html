Website basis

Nu je een bestandje hebt gemaakt, wordt het tijd om er iets in te stoppen. Er is een standaard basis voor HTML, we beginnen met de basis gewoon plakken. Later in het blok voor HTML leren we wat het betekent.

Dit is de basis voor elke HTML pagina: 


        <!DOCTYPE html>
        <html>
          <head>
            <meta charset="utf-8">
            <title> Website titel </title>
          </head>
          <body>
            <h1> Kopje </h1>
            <p> stukje tekst </p>
          </body>
        </html>

Opdracht

  1. Kopieer de basis code van hierboven en plak deze in het bestand wat je hebt gemaakt.
  2. Sla de veranderingen op met ctrl en s.
  3. Open de map waarin je jouw website hebt opgeslagen.
  4. Klik op de website die in de map staat en kijk naar het resultaat!
  5. Je kunt de website altijd herladen met de F5 knop. Dan hoef je hem niet opnieuw te openen.
