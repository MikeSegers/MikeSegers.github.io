Soms is het handig om een tabel te maken. Deze tabellen kun je gebruiken voor bijvoorbeeld een sportclub zou je bijvoorbeeld kunnen denken aan roosters of wedstreid uitslagen. Of misschien denk je eerder aan een webwinkel waar je het overzicht van jouw bestelling kan zien. Veel dingen waar je wilt dat de inhoud goed uitgelijnd is wordt gemaakt in een tabel.
Een tabel in HTML bestaat uit minstens 3 tags, maar dit kunnen er meer worden. Eerst heb je de table tag, hiermee vertel je dat je een tabel gaat maken. Dan maak je rijen voor jouw tabel, deze worden gemaakt uit tr tags, dit staat voor table row. Als laatst vul je de rijen met cellen. Hiervoor heb je de td tag wat voor table data staat, ofterwijl de inhoud. Hoe dit samenwerkt kun je zien in de foto hiernaast, met de html code die daar bij past. In deze tabel hebben we drie rijen en twee kolommen.

Naast de drie meest belangrijke tags voor de tabel heb je ook nog een speciale th tag. Deze gebruik je om het kopje van de rij of kolom aan te geven. Het staat voor table header. De th tag vervangt in dit verhaal de td tag.

Opdracht

Maak een tabel met minstens twee rijen en drie kolommen op jouw website. Zorg ervoor dat je in de bovenste rij de th tag gebruikt.
