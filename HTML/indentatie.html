Indentatie

We kennen nu echt al een heleboel tags die je kunt gebruiken om jouw website te vullen. Nu wordt het tijd om te leren hoe je de tags correct neerzet zodat het beter te lezen is. Dit doen we door middel van indentatie. De indentatie is hoeveel ruimte er zit voordat er tekst komt vanaf de linker kantlijn. Deze ruimte vullen we door middel van de tab knop links op het toetsenbord.
Elk voorbeeld die je tot nu toe gezien hebt maakt gebruik van indentatie. Je zult daarom waarschijnlijk in jouw bestanden al redelijk wat lege ruimte aan de linkerkant tegen komen.
De algeme regel is als volg: Na het openen van een tag voeg je een tabje meer toe tot je hem weer sluit.
Maar de regel zou geen regel zijn als we er geen uitzonderingen op zouden hebben. De uitzondering is gelukkig niet willekeurig, maar vraagt wel om weer een beetje theorie.

Verschillende type tags

Er zijn twee type tags in html block en inline tags.

type uitleg voorbeelden
inline Deze type tags worden niet op een nieuwe regel begonnen, en is slechts zo breed als nodig is. a, br, b, i, img, button, span
block Deze type tags beginnen netjes op een nieuwe regel, en maken gebruik van indentatie. Ze worden zo breed als het maar kan. p, div, h1-h6, main, section, footer, header, table, aside, header, ol, ul, li

Wat je dus goed moet onthouden is het volgende:
Na elke block tag gebruik je indentatie om er voor te zorgen dat het bestand leesbaar is
Nog een voorbeeld om het af te leren: 


        <!DOCTYPE html>
        <html>
          <head>
            <meta charset="utf-8">
            <title> Website titel </title>
          </head>
          <body>
            <h1> h1 is een block tag </h1>
            <p>
              De p is ook een blocktag, maar <b>de b</b> tag niet.
              html, head en body dan weer wel, maar img weer niet!
            </p>
          </body>
        </html>

Opdracht

  1. Neem onderstaande code over in een nieuw html bestand genaamd indentatie.html. Hierin is de indentatie weg gehaald
  2. Voeg alle indentatie weer toe op de juiste plekken door middel van de tab toets.
  3. Voeg een link toe naar indentatie.html aan de lijst van navigeren opdracht 2.

<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title> Website titel </title>
</head>
<body>

</article>
</main>

</html>
