Navigeren

We gaan voor dit onderdeel nog een paar tags bespreken die handig kunnen zijn voor het maken van jouw website. Eerst kijken we hoe we naar andere pagina's kunnen navigeren.

Opdracht 1

Maak een nieuwe HTML pagina aan, kijk eventueel nog even terug naar het maken van een bestand. Vergeet ook niet om je aan de naamgevings regels te houden en als laatst natuurlijk de basis structuur te gebruiken voor jouw website.

Linken

Hierboven zie je in het roze een link die je leidt naar de eerste theorie die we behandeld hadden voor het maken van de website. Terug linken naar een pagina is een manier om mensen tussen verschillende pagina's te laten navigeren, en wordt daarom ook als basis gebruikt voor de menu balken van websites.
De link maken we met behulp van de <a> tag. Deze tag bestaat uit drie onderdelen die we gaan bekijken aan de hand van het volgorde: 


        <a href="https://www.google.com/"> dit is een link </a>
  1. Aller eerst hebben we natuurlijk de tag zelf. Deze openen we netjes en sluiten we op het laatst af.
  2. Tussen de openingstag en de sluitingstag staat wat de tekst is die we laten zien op de website. In het voorbeeld zul je op de website zelf dus alleen de tekst dit is een link zien.
  3. Als laatst hebben we in de a tag een attribuut genaamd href. Je ziet dat deze een beetje lijkt op de src van de img tag. De href wordt gebruikt om te vertellen waar we naartoe linken. Dit kan een bestaande website zijn zoals in het geval van het voorbeeld. Ook kan je een pagina die je zelf hebt gemaakt linken, daarvoor staat hieronder een voorbeeld. Dit werkt ook het zelfde als bij de foto's! 

        <a href="tweedePagina.html"> dit is een link </a>

Lijstjes maken

Bij het maken van navigaties wordt er meestal gebruik gemaakt van een lijst. HTML heeft twee type lijsten die gebruikt kunnen worden.

  1. Een gesorteerde lijst
  2. Een niet gesorteerde lijst

Deze lijsten werken praktisch het zelfde, het enige wat je veranderd is de tag die je gebruikt voor het maken van de lijst. Eerst kijken we naar een voorbeeld van een gesorteerde lijst zoals je hierboven ziet. De tag die hiervoor gebruikt wordt is <ol> dit staat voor ordered list. Hierbij worden nummers gebruikt om de volgorde aan te geven. 


        <ol>
          <li> dit is het eerste element </li>
          <li> dit is het tweede element </li>
        </ol>

Je ziet in het voorbeeld dat er gebruik wordt gemaakt van de <li> tag om de elementen in te voegen. Hierdoor weet HTML welk nummer er toegevoegd moet worden.
De niet gesorteerde lijst gebruikt van de tag <ul> dit staat voor unordered list. Verder werkt hij echt precies het zelfde, kijk maar naar het tweede voorbeeld. 


        <ul>
          <li> dit is het eerste element </li>
          <li> dit is het tweede element </li>
        </ul>
