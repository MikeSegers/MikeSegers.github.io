Metadata

In de standaard HTML die we als eerst hebben geleerd, zat al een basis voor de head. We hebben geleerd dat dit alle informatie is die de browser nodig heeft om de website goed te laten zien. In de basis zit de charset wat vertelt welke tekens we allemaal begruiken. Daarnaast hebben we ook de title die gebruikt wordt om de naam van het tabblad te veranderen. Naast de titel van het tabblad zie je ook een klein logo. Dit kleine logo naast de naam van het tabblad heet een favicon. Deze wordt geladen met een link in de head. Op deze website zie je verder ook nog een stijling. Er zijn kleuren, de uitleg heeft een randje er om heen, begrippen zijn gekleurd etc. Dit wordt ook gedaan met een link naar de stijl in de head.
De head voor deze pagina ziet er bijvoorbeeld uit zoals de code hieronder. Je ziet hier dat er gebruik wordt gemaakt van de relatieve links zoals we hebben geleerd bij mappen. 


        <head>
          <meta charset="utf-8">
          <title> Metadata </title>
          <link rel="stylesheet" href="../opmaak.css">
          <link rel="shortcut icon" type="image/x-icon" href="../../favicon.ico"/>
        </head>

Opdracht

  • Voeg een favicon toe aan jouw website. Je kunt zelf een png zoeken, anders kun je ook het favicon van deze website gebruiken: https://raw.githubusercontent.com/isw-tld/isw-tld.github.io/main/favicon.png
  • Zorg ervoor dat je jouw website een eigen titel geeft.
