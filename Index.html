Jouw website!

Maak met behulp van de geleerde theorie, de indeling die je hebt gemaakt voor structuur en het kleurschema wat je hebt bedacht bij kleuren een website van minstens 1 eigen pagina. Het onderwerp mag je zelf weten en hoe je de inhoud doet ook. Zorg ervoor dat ik minstens het volgende kan terug vinden:

  • Jouw website is gemaakt volgens de layout die je hebt bedacht in opdracht structuur
  • Het kleuren schema wat jij hebt gemaakt in opdracht kleuren is volledig verwerkt in de website
  • Een menubalk met links naar de andere pagina's van de opdrachten
  • Een foto die je hebt opgeslagen in de map
  • Een lijst, of een tabel met inhoud die past bij de website
  • Correcte metadata voor de website
  • Correcte indetatie voor zowel de HTML als de CSS
Deze website is met liefde gemaakt door TLD en SGM. Is er iets mis? Mail dan met je schoolmail!
De kennis voor de theorie komt van bronnen als W3Schools en bittere ervaring. HTML en CSS is vooral veel oorzaak en gevolg zien. Je verandert iets kleins, en kijkt wat er gebeurt. Zo leer je het beste om alles zelf te stijlen.
