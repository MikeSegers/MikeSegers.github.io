Stijl toevoegen

We hebben nu een heleboel losse stukjes inhoud geleerd die je kunt gebruiken voor het maken van een website. Het enige wat ons nu dan nog te wachten staat is het stijlen van de website. Hiervoor zijn drie manieren die gebruikt kunnen worden.

Inline CSS

De makkelijkste manier van CSS toevoegen is inline. Hier kun je met het style attribuut aan elke losse tag zelf stijl toevoegen. Bijvoorbeeld als je een paragraaf met rode tekst wilt zou je in de openingstag het attribuut style="color: red;" zetten. De code ziet er dan zo uit 


        <p style="color:red;">
        Een geweldige rode tekst
        </p>

Als je deze manier gebruikt moet je voor elke tag, elke keer los de CSS toevoegen. Dit wordt echt een chaos!

Internal CSS

Je kunt daarom ook in de head een style tag gebruiken om in het algemeen de kleur aan te passen. Het voordeel hiervan is dat je niet elke p los rood hoeft te maken. In de style tag gebruik je de taal CSS. Deze wordt opgebouwd op de volgende manier:

selector {eigenschap: waarde;}.

Je kunt zo veel eigenschappen en waardes toevoegen binnen de accolades als je nodig hebt. Sluit ze wel altijd netjes af met een puntkomma. De selector is de tag die je wilt veranderen, bijvoorbeeld de p. De eigenschappen die een tag heeft verschillen, maar zijn per tag terug te vinden op W3Schools. Voor dit voorbeeld werken we met de eigenschap tekst kleur, deze geef je aan met color. De waarde die we hieraan toekennen is red voor rood.
De interne CSS zet je als onderdeel van de head neer en ziet er dan als volgt uit: 


        <style>
          p { color: red; }
        </style>

Je kunt hiermee al een groot deel van de chaos weghalen, het nadeel komt vooral als je meedere pagina's hebt die allemaal dezelfde stijl nodig hebben. Stel je voor je hebt 10 pagina's en je wilt de tekst nu blauw maken. Je kunt dan de waarde blue opschrijven, maar dit moet je voor alle 10 de pagina's los doen!

External CSS

Uiteindelijk is de beste optie om externe CSS te gebruiken. Hiervoor maak je één stijl bestand aan waar je alle stijling in verwerkt. Deze koppel je dan in de head aan de pagina, en dan kun je alle pagina's tegelijk mooi maken. In de head voeg je dan de volgende code toe: 


        <link rel="stylesheet" href="opmaak.css">

Let op dat als je de naam van het bestand anders maakt, of je zet de css in een mapje dat je de link van de href correct aanpast zoals je hebt geleerd bij het gebruiken van mappen.

Het bestandje opmaak.css vul je vervolgens weer met de CSS code zoals ook uitgelegd in Internal CSS.

Opdracht

  1. Ga naar Notepad++ en maak een nieuw bestand aan met ctrl en n.
  2. Klik op ctrl, alt en s om het bestand op te slaan.
    1. Zoek naar de map die je hebt gemaakt voor jouw website.
    2. Noem het bestand opmaak.css
    3. De bestandstype moet een cascade style sheet file zijn.
  3. Voeg in opmaak.css de code p { color: red; } toe en sla het op met ctrl en s. Je mag de kleur ook veranderen.
  4. Verbind de CSS met jouw index pagina zoals je hebt geleerd bij External CSS.
