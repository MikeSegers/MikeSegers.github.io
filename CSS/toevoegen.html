Stijl toevoegen

We hebben nu een heleboel losse stukjes inhoud geleerd die je kunt gebruiken voor het maken van een website. Het enige wat ons nu dan nog te wachten staat is het stijlen van de website. Hiervoor zijn drie manieren die gebruikt kunnen worden.

External CSS

Uiteindelijk is de beste optie om externe CSS te gebruiken. Hiervoor maak je één stijl bestand aan waar je alle stijling in verwerkt. Deze koppel je dan in de head aan de pagina, en dan kun je alle pagina's tegelijk mooi maken. In de head voeg je dan de volgende code toe: 


        <link rel="stylesheet" href="opmaak.css">

Let op dat als je de naam van het bestand anders maakt, of je zet de css in een mapje dat je de link van de href correct aanpast zoals je hebt geleerd bij het gebruiken van mappen.

Het bestandje opmaak.css vul je vervolgens weer met de CSS code zoals ook uitgelegd in Internal CSS.

Opdracht

  1. Open het bestand style.css in Replit
  2. Voeg in style.css de code p { color: red; } toe en kijk wat er veranderd aan je website als je op run duwt. Je mag de kleur ook veranderen.
  3. Verbind de CSS met jouw index pagina zoals je hebt geleerd bij External CSS.
