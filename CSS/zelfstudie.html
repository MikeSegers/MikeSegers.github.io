Nog zo veel meer

Er zijn nog een heleboel onderdelen van CSS die we niet hebben besproken. Dat komt natuurlijk omdat er echt heel veel kan, maar we niet alles gelijk moeten uitleggen anders word je overspoelt met te veel informatie. Voor deze opdracht ga je dus zelf kijken naar dingen om toe te voegen. Het beste is om W3Schools te gebruiken, omdat dit een hele grote website is die je voor veel meer dan alleen websites maken kan leren gebruiken. Andere websites zijn natuurlijk ook toegestaan. Hier zijn enkele voorbeelden van dingen die je nog kan opzoeken, maar als jij wat anders wilt, voel je vrij! Het is tenslotte zelfstudie.

  • Tabel opmaken
  • Orderd list opmaken
  • Animaites invoegen
  • Font veranderen
  • Afbeeldingen stijlen
  • Gradients toevoegen

Opdracht

  1. Kies twee onderdelen die we nog niet eerder geleerd hebben van W3schools.
  2. Vermeld op jouw index wat je hebt toegevoegd met het kopje "Zelfstudie".
  3. En verander het in de CSS zodat het werkt.
