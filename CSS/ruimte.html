Ruimte en lijntjes

In HTML zet je alles van jouw website in verschillende tags. Deze tags kun je zien als een soort blokje wat om de inhoud komt te staan. Bijvoorbeeld de p is een blokje om de tekst heen op jouw website. CSS gebruik je om deze blokjes te stijlen. We hebben het al gehad over het veranderen van de tekst op de website, nu gaan we kijken hoe we de ruimte van het blokje kunnen veranderen. Hiervoor moeten we eerst een lijntje om het blokje tekenen zodat we makkelijker erover kunnen praten.

De border-style heb je nodig om het randje aan te zetten, daarna kun je met border-width, border-color en border-radius jouw randje veranderen. De knoppen onderaan deze pagina zijn bijvoorbeeld gemaakt door een randje erom heen te zetten. Je kunt ook alleen een randje aan de bovenkant, onderkant, linker of rechter kant zetten met border-top-style, border-bottom-style, border-left-style, border-right-style. Deze vervangen dan de border-style eigenschap.

Opdracht 1

Voeg hiervoor de volgende code toe aan jouw website:


<p id="ruimte">
  In dit stukje leren we hoe we de ruimte van onze blokken kunnen veranderen.
  We begruiken dit stuk tekst om het te begrijpen en bewerken.
</p>

En daarna deze code aan jouw CSS. 


#ruimte {
  border-style: solid;
  width: 30%;
}

Ruimte toevoegen

Er zijn twee manieren waarop we ruimte kunnen toevoegen, het lijntje hebben we toegevoegd zodat we goed kunnen zien waar de ruimte zit. Het binnenste hokje wordt gebruikt voor de inhoud die je hebt staan. Dit kan tekst zijn, een foto of iets anders wat er tussen de tags staat. Daarom heen heb je padding dit is de ruimte tussen de inhoud en het randje in. Dan heb je het randje zoals we hierboven hebben geleerd. Buiten het randje zit de margin deze wordt gebruikt om ruimte tussen verschillende blokken te maken. Voor margin en padding heb je ook weer de boven, onder, links en rechts. Bijvoorbeeld de padding aan de bovenkant kun je veranderen met de padding-top eigenschap.

Opdracht 2

Margin heeft een interesante instelling die je kan gebruiken om elementen in het midden te krijgen. Dit doe je door de margin links en rechts op auto in te stellen. Verander de css code van #ruimte zodat de tekst met id ruimte in het midden van de pagina komt te staan met behulp van de margin.
