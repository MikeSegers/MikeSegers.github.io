Kleuren kiezen

Elke website, elk product heeft een bepaald kleuren schema waarmee je vaak gelijk kan herkennen wat het is. Deze website heeft bijvoorbeeld blauw als hoofdkleur met daarbij roze als secundaire kleur en groen als tertiaire kleur. Dit kleurenschema heb ik bedacht met behulp van een color picker website. Andere websites die je kan proberen zijn:

  • color picker website 1
  • color picker website 2
  • color picker website 3
Je kunt als je daar behoefte aan hebt ook kijken naar de betekenis van de kleuren als je een specifiek idee wilt overbrengen via een website over kleur betekenis

Opdracht 1

  1. Eerste kies je een hoofdkleur uit die jij mooi vind, of die goed bij een onderwerp past.
  2. Daarna kun je in het dropdown menu er onder verschillende opties zien, zoals complementary, triadic en analogous. Kies een van de opties om de secundaire en misschien tertiaire kleur voor jouw website te bepalen.
  3. Dan kun je met neutrals, shades, tints en tones de verschillende subkleuren bepalen.
  4. Exporteer alle kleuren en ze deze in jouw opmaak.css. Je kunt hiervoor de HEX of RGB waardes als beste kiezen de reden hiervoor wordt duidelijk in het volgende stukje.
    Als je graag een voorbeeld hiervoor wilt kun je kijken naar de CSS van deze website ook hier is bovenin een lijstje gemaakt van de kleuren met daarbij behorende tints en shades. Om er voor te zorgen dat dit niet gezien wordt als foute CSS code kun je de informatie verberen met een comment. Om dit te doen zet je voor de tekst een /* en na de tekst */. Ook dit kun je zien in het CSS bestand van deze website.

Kleuren toevoegen

Nu we kleuren hebben gekozen kunnen we deze gebruiken om bijvoorbeeld de tekst kleur te veranderen. Je kunt dit doen op verschillende manieren, hieronder zie je een lijst met opties om de kleur op rood in te stellen. Hier zie je ook terug waarom je de rgb of de hex waardes nodig had bij de vorige opdracht.

  • p {color: red;}
  • p {color: rgb(255, 0, 0);}
  • p {color: #FF0000;}
  • p {color: rgba(255, 0, 0, 1);}

Kleuren handig gebruiken

Niet elke kleur heeft een naam, hierdoor zijn de rgb of hex waardes een stuk betrouwbaarder, maar het is ook niet prettig om elke keer de hex of rgb waarde te moeten opzoeken om deze te gebruiken. Gelukkig is er een manier om de kleuren een naam te geven voor CSS. Hieronder is het voorbeeld voor de lijst met kleuren voor deze website. 


:root {
  --main: rgb(34, 132, 220);
  --main-s2: rgb(26, 99, 165);
  --main-s4: rgb(17, 66, 110);
  --main-t2: rgb(89, 163, 229);
  --main-t4: rgb(117, 178, 233);
  --main-t6: rgb(172, 209, 242);
  --main-t8: rgb(200, 224, 246);
  --main-t10: rgb(227, 240, 251);
  --accent1: rgb(220, 34, 132);
  --accent1-s2: rgba(220, 34, 132, 0.8);
  --accent2: rgb(132, 220, 34);
  --accent2-s2: rgba(132, 220, 34, 0.8);
  --accent2-s4: rgba(132, 220, 34, 0.6);
}

Hierin is de --main, --accent1 en --accent2 als basis gedefineerd. Besluit ik over een jaar dat ik liever andere kleuren wil gebruiken, dan hoef ik alleen in die lijst de rgb kleuren aan te passen en gaat de rest vanzelf!
Je gebruikt de kleuren op de volgende manier: p {color: var(--main);} hiermee verwijs je naar de kleur --main die je in de :root hebt aangemaakt.

Opdracht 2

  1. Ga naar jouw opmaak en maak de lijst met kleuren zoals hierboven aangegeven, je mag zelf de kleuren kiezen.
  2. Gebruik een van de kleuren om de h1 tags een andere kleur te geven.
