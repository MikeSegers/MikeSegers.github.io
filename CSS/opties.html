Onderscheiden: class en id

Tot nu toe hebben we elke keer voor een bepaalde tag ervoor gezorgt dat alles dezelfde opmaak krijgt. Met natuurlijk de uitzondering van de stukken die jullie van mij gekregen hebben. In dit onderdeel leer je waarom en hoe die uitzonderingen gemaakt worden door middel van class en id. Beide onderdelen zijn een attribuut die je toe kan voegen aan een tag. bijvoorbeeld <p class="col1"> of <p id="ruimte">.

Class

De class attribuut wordt toegevoegd aan alle onderdelen die een bepaalde eigenschap moeten hebben. Bijvoorbeeld, je kunt een class links maken, die je gebruikt voor alle tags die links uitgelijnt moeten worden. Of bijvoorbeeld een class center voor alle tekst die je in het midden wilt hebben. Deze classes kun je ook combineren zoals dit: <p class="links center">. Hiermee krijgt deze tekst in de p tag zowel de eigenschappen van links, als van center. In het CSS bestand krijgen zet je een . voor de naam van de class om de eigenschappen te defineren. Bijvoorbeeld: 


        .links {
          float: left;
          width: 20%;
        }
        .center {
          text-align: center;
        }

Opdracht 1

  1. Voeg de CSS code toe van hierboven aan jouw CSS.
  2. Voeg de HTML code van hieronder teo aan een van jouw websites.
  3. Voeg de classes links center toe aan beide p tags.

          <p>
            Dit is een stukje tekst die we hebben die straks in het midden is,
            Daarnaast komt het blokje tekst aan de linkerkant voor 20% van het scherm.
            Het is best handig om de classes toe te voegen zodat je snel dingen kan stijlen.
          </p>
          <p>
            Hierna gaan we kijken naar ID, je weet het nog niet, maar die gaan we toevoegen aan deze
            paragraaf. Hiermee zorgen we dat alleen deze een bepaalde achtergrond kleur krijgt.
          </p>

ID

De ID is ook een attribuut, en werkt eigenlijk op een gelijkende wijze als de class. De ID wordt alleen gebruikt om specifieke elementen aan te duiden. Dit wordt namelijk ook gebruikt als je bijvoorbeeld javascript toevoegt. Dit is een programmeertaal die je kunt gebruiken om de website interactief te maken. Met Javascript kun je een element met specifiek ID aanspreken om hem bijvoorrbeeld te verstoppen, of te laten bewegen over het scherm. Dit kan een begin zijn van een spel?
De ID kun je ook stijlen net als de class. Hiervoor gebruik je de hashtag voor de naam om hem als selector te gebruiken. Je krijgt dus de combinatie van <p id="specifiek"> met #specifiek { ... }

Opdracht 2

  1. Voeg de CSS code van hieronder toe aan jouw CSS.
  2. Zorg dat de background color een van jouw kleuren is, met behulp van de var.
  3. Voeg de id tweede toe aan de laatste p tags.

          #tweede {
            background-color: var(--main);
          }
